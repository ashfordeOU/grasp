📖 Grasp Team Dashboard — Guide
🚀 Getting Started
Add a repoPaste owner/repo or a full GitHub URL, press Enter or click + Add Repo
Load examplesClick "Load examples" to pre-populate with popular open-source repos
GitHub tokenOptional — paste a PAT for 5,000 req/hr and private repos. Shared with Grasp: set it in either app and both pick it up
Saved reposYour repo list and token are saved automatically in localStorage — they persist across browser sessions
Open in GraspClick any repo name in the table to deep-dive into the full dependency graph in index.html
📊 What the Dashboard Shows
Health Score0–100 score per repo: A (90+) · B (75+) · C (60+) · D (45+) · F (<45)
Grade badgeColor-coded A–F grade for at-a-glance quality. Green = healthy, red = needs attention
Issues countTotal architecture issues: circular deps, dead code, high coupling, security findings
Circular depsNumber of circular dependency chains detected across the repo
SecurityHardcoded secrets, SQL injection risks, dangerous patterns found
LayersArchitectural layers detected (UI, Services, Data, Utils, Test, Config…)
Score bars chartVisual ranking of all repos by health score — longest bar = healthiest
Issue breakdownAggregate issue counts across all repos, grouped by issue category

🔒 Privacy
100% browser-basedAll analysis runs in your browser — no backend, no data sent anywhere
GitHub APICalls go directly to api.github.com. Without token: 60 req/hr. With PAT: 5,000 req/hr
Token storageYour GitHub token is stored in localStorage only — it never leaves your browser. Shared with Grasp so you only need to set it once
